Interests

技術系

  • Webアプリケーション開発

    • 授業で学んだことを踏まえ、最近ではWebアプリを制作する機会が増えました。
    過去にログイン機能の実装やDBを用いたデータの記録・閲覧、地図を用いたデータの可視化などを行っています。

  • テキストマイニング

    • 研究で使用したことを機に興味を持ちました。
    エンベディングを用い、文章間の類似度計量などを主に取り組んでいます。

  • XAI(説明可能なAI)

    • 近年生成AIなどの進化が著しい中、人間が考えたような、もしくそれ以上の結果を出力することに驚かされることが多くなりました。
    しかしそれらの多くは、その出力になった根拠を説明することが困難となっています。
    出力に行き着くまでの過程を説明可能にすることは今後ますます重要になると考えており、それらの手法について調査しています。

  • Web API

    • 研究の一環として一度用いて以降、別のサーバーと情報を共有できる点に興味を持ちました。
    まだ経験や技術は浅く、勉強中の分野です。

分野

  • 交通

    • 小学校に入るまでは自動車が好きで、それからは鉄道が好きになりました。 いわゆる「乗り鉄」と呼ばれる部類です。
    また現在大学ではサイクリングクラブ(元部長)に所属しており、交通関係は全般的に興味があります。

  • 旅行

    • 上記の通り「乗り鉄」なので様々な鉄道に乗るのが好きで、そこから旅行好きにもなりました。
    年に5,6回は1人旅行も含め、全国様々なところを旅行します。 10代のうちに47都道府県制覇するという目標を決め、無事達成しました。
    このようなことから「MaaS」に関して非常に興味があり、現在研究を進めています。

  • 地理・地図

    • 先の2つに関連する分野ですが、地理や地図に興味があります。
    実際に町へ出て地形の成り立ちを考えたり、全国の地名の由来を調べてまとめたり、明治時代などの古地図を閲覧することもあります。
    また既存のアプリケーションを用いた地図上への可視化も多く行います。

  • 防災・減災

    • 高校生のときに私は地元の大阪で大きな地震・台風に遭い、町がボロボロになっていくのを目の当たりにしました。 また最近では毎年のように全国で大きな災害が発生しています。
    災害で苦しむ人が減るように、また災害にあってからもなるべく普通の生活が出来るようにならないかと考えています。
